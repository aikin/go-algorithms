# Valid Anagram

> Given two strings s and t , write a function to determine if t is an anagram of s.



**Example1:**

```
输入: s = "anagram", t = "nagaram"
输出: true
```

**Example2:**

```
输入: s = "rat", t = "car"
输出: false
```
